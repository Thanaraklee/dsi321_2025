# Prepare
1. Create environment
```bash
python -m venv .venv
```

2. Activate your environment
    - Windows
        ```bash
        source .venv/Scripts/activate
        ```
    - MacOS & Linux 
        ```bash
        source .venv/bin/activate
        ```
3. Run script
```bash
bash start.sh
# or
./start.sh
```
